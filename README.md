# City Morgue React

React-сайт о группе City Morgue: биография, таймлайн, распад, альбомы, истории треков и кнопки прослушивания 30-секундных превью через iTunes/Apple Music.

## Что использовано

- React + Vite
- `useState` для активного трека и фильтра по годам
- props в компонентах `TrackCard`, `AlbumCard`, `MemberCard`
- `lucide-react` для иконок
- легальные preview URL из iTunes API

## Запуск

```bash
npm install
npm run dev
```
Или же
https://zemnyux.github.io/city-morgue-site/

Если порт `5173` занят, Vite автоматически откроет следующий свободный порт.

npm run lint
